About the Product

GoldenSherpa is a platform designed to help families find and connect with residential healthcare facilities — senior living communities, assisted living facilities, and memory care centers — for their loved ones. The platform digitizes what was previously a fragmented, phone-and-paper process into a seamless digital experience.

The product allows placement advisors, families, and care facility staff to schedule tours, conduct video call consultations, exchange documents, and track placement progress — all in one place.

Key Features Built

  • Event scheduling system for facility tours and consultation appointments
  • Video call integration for remote family-facility consultations
  • Placement tracking dashboard for advisors managing multiple family cases
  • Facility search and matching based on care needs, location, and budget
  • Document exchange system for medical records and facility agreements
  • Notification system for appointment reminders and status updates
  • Multi-role system — families, placement advisors, and facility managers
  • AWS S3 integration for secure document and media storage

Technical Highlights

The scheduling system was built as a flexible event management module on Django, supporting recurring availability slots, timezone-aware scheduling, and conflict detection. The video call feature was integrated using a third-party WebRTC service, embedded within Angular components.

AWS was used for S3 document storage, SES for email notifications (appointment confirmations, reminders), and EC2 for deployment. The Angular frontend provided a smooth single-page experience for the complex multi-step placement workflow.

Impact

  • Digitized residential healthcare placement workflows for US market
  • Enabled remote family consultations reducing geographic barriers
  • Streamlined advisor workflows for managing multiple active placements simultaneously
